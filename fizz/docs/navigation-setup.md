# Fizz Navigation — Subscriptions Link

Add a customer-facing link to the subscription portal:

1. **Shopify Admin → Online Store → Navigation**
2. Open the **Support** menu (used by the footer)
3. Add menu item:
   - **Name:** My CO₂ refills
   - **Link:** `/account?view=subscriptions`
4. Save

Customers with orders can manage on-demand Fizz Charge refills from this page.
