# Arctic Light

**Color theory:** Inverted light-surface mode — lime accent on warm paper hero, ink text (roles swapped vs dark presets).

| Token | Value |
|-------|-------|
| Accent | `#9ed12f` |
| Ink (surface) | `#f2efe7` |
| Paper (text) | `#080a0d` |

**Section intent:** Light hero background; statement band flips to dark ink with paper text. Footer returns to dark for dramatic close.

**Availability:** Not a native Shopify Theme style (5-preset platform limit). Select **Arctic Light** from Colorway preset dropdown.

**Contrast:** Review nav `mix-blend-mode` on light hero — may need Normal blend or nav sync tint.
